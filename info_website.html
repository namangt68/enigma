What are the learning outcome form this project)



  • Python We used the python programming language in this project and it was a joy to work with. We used several python modules and libraries, like hashlib, which we used to hash the file contents to their sha1sum to store in the git object store.
  • Multiprocessing We used to initiate a new process to open the editor.
  • Working with tempfilesThis project also taught us how to work with tempfiles and their importance. The commit message was first stored in a tempfile and from their on we stored it in the message file.


  • We avoided multi threading in favour of multiprogramming , to avoid shared data and the problems that come with it. We used several processes with no shared data.
